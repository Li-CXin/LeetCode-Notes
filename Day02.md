# Day02 数组2

## 209. Minimum Size Subarray Sum (长度最小的子数组)
一开始在尝试自己回忆滑动窗口的操作，大致回忆起是先用右指针扩大子数组的范围，再用左指针缩小范围。思路本身并不复杂，但在实现上有些困难，于是去看了讲解才回忆起原来是用一个外循环的for和一个内循环的while实现的。另外复习了python中无限大的写法：float('inf')

## 59. Spiral Matrix II (螺旋矩阵II)
一开始看到这个题有关循环，所以想用快慢指针，但发现怎样都解不出来去看了讲解，才发现原来是一道模拟题，于是又按照模拟题的思路，在到达四个角元素之后进行条件判断——当到达边界或者该位置已经有元素时，转变方向。但在实现过程中，如何模拟“转弯”有些困难，即使有正确的思路，一口气写出正确代码也很不容易，是一道非常考验条件设定能力的题目，需要有意识地对循环不变量——即向量区间进行设计。
