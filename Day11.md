## 150. Evaluate Reverse Polish Notation（逆波兰表达式求值）
> **Tags**: [`#栈与队列`](/search?q=栈与队列)
逆波兰表达式：是一种后缀表达式，所谓后缀就是指运算符写在后面。平常使用的算式则是一种中缀表达式，如 ( 1 + 2 ) * ( 3 + 4 ) 。该算式的逆波兰表达式写法为 ( ( 1 2 + ) ( 3 4 + ) * ) 。本题的逻辑并不复杂，依然是利用栈的先进后出原理来计算后缀表达式，但需要注意的是LC中的题目要求为除法应““向零截断””，而讲解中写的则是“只保留整数部分”，而“//”是向下取整。对于正数没问题，但对于负数，如-1 // 10 在 Python 中结果是 -1。而按照LC的题目要求则 int(-0.1)，结果应该是 0。
